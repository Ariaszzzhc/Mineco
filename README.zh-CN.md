## Mineco 是什么？

Mineco 是一个原生桌面应用，让 AI 编程助手直接在你的项目目录里工作。读代码、改文件、跑命令 —— 全在一个窗口里完成。

打开文件夹，直接开干。

## 核心能力

- **文件操作** — AI 直接在工作区读写文件。不是给建议，是直接改。
- **终端命令** — 执行命令、查看输出、定位错误，一个对话搞定。
- **深度思考** — 支持 Extended Thinking，复杂问题先推理再动手，思考过程可展开查看。
- **子代理** — 派生子代理并行处理子任务，各自独立运行，有独立的上下文和步数限制。
- **实时流式响应** — 基于 SSE 的实时流式输出，文本、工具调用、执行结果即时呈现。
- **自动会话标题** — AI 根据首条消息自动生成会话标题。
- **富文本渲染** — Markdown 渲染，Shiki 语法高亮，可折叠思考块，工具调用卡片。
- **Provider 系统** — 支持任意 OpenAI 兼容 API，内置智谱 AI 支持，可自定义接入端点和模型。
- **以工作区为中心** — 会话和配置都绑定在项目目录上。
- **桌面 + Web** — 可作为 Tauri 原生桌面应用运行，也可启动 Web 模式通过浏览器访问。

## 快速开始

### 环境要求

- Node.js >= 25
- pnpm 10.29

### 安装与运行

```bash
pnpm install && pnpm start
```

在 **设置** 里填入 API Key，打开一个文件夹，开始写代码。
